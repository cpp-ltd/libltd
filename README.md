## libltd

A simple C++ framework and library.

### Installation

Create a build directory and navigate to it for an out-of-source build:
```sh
mkdir build && cd build
```

Run CMake to generate build files:
```sh
cmake ..
```

Build _libltd_:
```sh
make
```

Install _libltd_ if needed:
```sh
sudo make install
```
